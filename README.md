# all_in_one

万能工具库

### compress (压缩/解压)

- `studio_zip.h`: zip 压缩解压 

### encode (编码)

- `studio_encode.h`: 编码转换


### file (文件)

- `studio_dir.h`: 目录的基本算法
- `studio_file.h`: 文件的基本算法


### geometry (矢量多边形)

- `studio.geo.h`: 矢量类型
- `studio.geo_algo.h`: 矢量算法


### json

- `studio_json.h`: json对象操作


### log

- `studio_log.h`: 日志记录


### singleton (单例)

- `studio_singleton.h`: 单例类


### string

- `studio_str_algo.h`: 字符串的基础算法
- `studio_regex.h`: 正则匹配

### system (系统后台)

- `disk_space.h`: 磁盘空间
- `background.h`: 后台程序


### time

- `studio_timer.h`: 计时器
- `studio_date.h`: 时间操作


### verify (校验)

- `studio_crc.h`: crc校验
